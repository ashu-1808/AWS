# CIDR (Classless Inter-Domain Routing)

```
CIDR format: x.x.x.x/n

Where:
n = number of network bits
Remaining bits = host bits
```

## Formula:

```
Total IPs = 2^(32 - CIDR)
Usable IPs = 2^(32 - CIDR) - 2
```

Note:

* 1 IP is Network ID
* 1 IP is Broadcast ID
  (Except /31 and /32 special cases)
